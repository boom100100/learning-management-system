Install:

Fork, then clone the repo:
```
git clone https://github.com/your-username/rails-portfolio-project.git
```

Install gems (see Gemfile for version of rails):
```
bundle install
```

Set up database:
```
rake db:create
```
Seed database:
```
rake db:seed
```
